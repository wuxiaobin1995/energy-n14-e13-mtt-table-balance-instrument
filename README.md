# 台式平衡康复训练仪-MTT 版

## 串口定义

- 第一位代表 x 轴距离，第二位代表 y 轴距离（单位：mm）
- 前、右为正，左后为负
- 0.1 秒发一次数据
- y 轴往前照射，x 轴往左照射
- x 轴（左右）单边走 60mm，y 轴（前后）单边走 90mm
